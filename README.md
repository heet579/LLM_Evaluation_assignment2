# LLM_Evaluation_assignment2

Evaluation of open-source LLMs regarding phishing susceptibility and bias using the DECODINGTRUST framework.

## Files Included
* `01_LLM_EVALUATION.ipynb`: Jupyter notebook containing the experimental setup and data collection pipeline across multiple models.
* `prompt_generation.csv`: The 1,000-sample dataset generated, evaluating multiple models on persona-based phishing email contexts.
* Reference research papers covering the DECODINGTRUST framework (`2306.11698v5.pdf` and `2311.12827v1.pdf`).
